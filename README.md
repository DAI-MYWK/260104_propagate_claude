# Propagate Inc. ホームページ

株式会社プロパゲートのコーポレートサイトです。

## 概要

WEB制作・マーケティングサービスを提供するPropagate Inc.の公式ホームページです。月額9,800円〜のホームページ制作、Shopify EC構築、広告運用、TikTok CM配信などのサービスを紹介しています。

## 構成

- **TOPページ**: `index.html`
- **サービスページ**: `web-design.html`, `shopify-design.html`, `subscription-ad.html`, `spark-buzz.html`
- **会社情報**: `company.html`
- **お問い合わせ**: `question.html`

## セットアップ

### ローカル環境での確認

```bash
# ローカルサーバーを起動
python3 -m http.server 8000

# ブラウザでアクセス
open http://localhost:8000
```

または、`index.html`をブラウザで直接開いて確認できます。

## 技術スタック

- HTML5
- CSS3（モジュラー構成）
- JavaScript（バニラJS）
- Unsplash（ダミー画像）

## デザイン原則

- デザインの4原則（近接・整列・コントラスト・反復）に準拠
- LPデザインルール・コーポレートサイトルールに従った構成
- 70:25:5のカラー比率
- Noto Sans JP + Montserratフォント使用
